How to Subscribe to an event

This is a tutorial for how to subscribe to events on the event busses.

1. Which event bus

First you need to figure out which event bus you are subscribing to. The Mod event bus is used for mod setup and registry events. The Forge event bus is used for everything else.

2. Method of subscribing

There are 3 options. You can either use bus.regsiter(MyClass.class) or bus.register(myobject) or @Mod.EventBusSubscriber on the event handler method's class.

Note: If you are subscribing a class, with either bus.register(Class) or @Mod.EventBusSubscriber, the event handler method should be static

3. Add @SubscribeEvent?

You should only use @SubscribeEvent when you are using the automatic event bus subscriber, @Mod.EventBusSubscriber.
