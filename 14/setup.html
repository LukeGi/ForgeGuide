Setting up a Modding Workspace

This guide will assume you are using Intellij IDEA. The steps are similar for other IDEs.

1. Download the MDk

Download the latest Forge MDK. Download Here

2. Extract into your folder

Now extract the MDK, ensuring that the build.gradle file is in ModFolder/build.gradle.

3. Import the gradle project into your IDE

Now open Intellij IDEA, and import the build.gradle file.

4. Edit files for your mod

Now open build.gradle and replace the version, group and archivesBaseName to match your requirements. Then edit the mods.toml found in resources/META-INF.

5. Generate run tasks

Now, you should have a gradle tab on the right, in which you can find the task genIntellijRuns. Run this task.

6. Create main mod class

Now create a class for you mod. This should be located in the group, and named the archivesBaseName you specified in your build.gradle. See the examplemod for an example of how to setup this main mod class. Note: the most important thing is the @Mod("modid") being above your class.

7. Run the game

The run task you ran earlier should have added a run config called 'runClient'. Run this and you should see that your mod has loaded with the game.
